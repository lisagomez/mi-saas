Giant metal numeral + accent underline + label/note — the impact KPI. Use three across.

```jsx
<KPIStat value="<45 s" label="Dictamen con ruta legal" note="−90% vs. el proceso manual" />
<KPIStat value="80%"   label="Cobertura" note="conceptos de gasto recurrentes (PM)" />
```

Pre-format `value` yourself (include `<`, `≥`, `%`, `s`). In an animated build, count the digits up and grow the underline width from 0 → 64%.
