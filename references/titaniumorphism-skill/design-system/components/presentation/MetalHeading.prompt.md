Metal-filled display heading — the titanium wordmark / scene-title move. Use for display roles only, on a graphite background.

```jsx
<MetalHeading text="Grafo" size="mega" align="center" />
<MetalHeading text="Trazabilidad total" size="h1" />
```

Sizes: `mega` (wordmark, 184px, weight 800, tracking −0.04em) · `h1` (scene title) · `h2`/`h3` (statements). Pair with `<Kicker>` above it. Never use for body text — body stays Manrope in `--text-hi`/`--text`.
