Frosted-glass plate for a single stat + caption. Use 2–3 in a row for a "cost / impact" beat.

```jsx
<GlassChip value="20–100" unit="%" label="multa sobre el impuesto, por una deducción mal sustentada" />
```

Must sit on `--surface-page` (or any graphite bg) for the backdrop blur to show. Keep captions to one short line. For headline KPIs that count up, use `KPIStat` instead.
