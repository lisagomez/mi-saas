Uppercase mono section kicker (wide tracking) — names the beat above a heading.

```jsx
<Kicker text="El diferenciador" />
<MetalHeading text="Trazabilidad total" size="h1" />
```

Always mono, always uppercase (the component handles casing). Steel by default; pass `color="var(--accent)"` only for a CTA kicker like "Hablemos".
